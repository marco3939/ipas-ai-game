🎯 任務目標

2026-05-23 IPAS AI 應用規劃師中級考試合格(70 分)。本場練習場依五大鐵律設計,專治「題目都背過但考試還是不會」「都做一樣的題目」「會做但講不出來」。

⚔️ 角色狀態(HP / MP / EXP / Level)

戰鬥模式有 RPG 三條值:

  • HP(血量):初始 100/100。答錯扣血(各 mode 8~12 + boss HP 加成)。歸 0 → 戰鬥失敗,可重來,不會永久死亡。
  • MP(法力):初始 50/50。用來發動技能(提示、消除錯選項、雙倍傷害等)。
  • EXP(經驗):答對累積,局末結算發放。
  • Level(等級):從 1 開始。升級條件:EXP ≥ expMax(初 100,每升 ×1.5 → 150 / 225 / 337…)。

升級獎勵:HP 上限 +20、MP 上限 +10、HP/MP 補滿、技能點 +1。

📈 EXP 怎麼賺

  • 答對題目 → 局末一次性結算(每模式公式不同,大致 10~30 EXP/題,難度與 combo 加乘)
  • BOSS 戰擊殺 → 額外 bonus EXP
  • 下鑽變化型答對 → 同題加成(鐵律 #1)
  • 注意:EXP 不是即時加,要打完一整局才給;中途離場可能損失

📅 SM-2 間隔重複系統

每題依答題表現,系統自動排下次該複習日:

  • 答對 + 流暢(高 EF):間隔變長(1 → 6 → 15 → 35 天遞增)
  • 答對 + 卡頓:間隔小幅延長
  • 答錯:重置,明天再考
  • 首頁「📅 今日複習」會顯示到期題數,清空就過關

📝 錯題本(鐵律 #1 治本)

  • 每答錯一題自動入冊
  • 連續答對 3 次 → 從錯題本移除(視為熟練)
  • 錯題本內每題都有變化型訓練(case_a/b/c 動態出題)
  • 進入錯題本 → 重做錯題 + 變化型,跨場累積熟練度

📜 五大鐵律(出題與機制原則)

  1. 錯題下鑽 — 答錯一題 = 看見一個知識盲點,系統強制針對同節點生成變化型再考,不讓你跳過
  2. 動態題庫(case_a/b/c) — 同題不同變數,治「都背過題目了還是不會」
  3. 不抄考古題 — 不直接搬 114-2 真題,自製改寫保留原概念但變題型
  4. 選項等長 — 消除「選最長那個就對」的作弊路徑(IPAS 真考也遵守)
  5. 來源忠實 — 所有節點 / 題目都對應官方 KB 白名單,不憑空造
