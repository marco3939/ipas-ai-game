為 IPAS AI 中級「AI 應用規劃師」考試準備互動式練習遊戲,涵蓋:

## 開發流程(9 階段,計畫先行,逐階確認)
0. 環境核對與目錄建構 ← **目前**
1. 範圍劃定(科一 / 科三 / 科二邊界)
2. 考古題模式分析(題型 / 高頻知識點 / 易混淆組合)
3. 知識節點抽取(分批處理,/clear 控管 context)
4. 5 種遊戲方案提案
5. 詳細設計
6. 題庫生成(`src/questions.json`)
7. 網站建置(本機 `python -m http.server 8000` 驗證)
8. 推送 GitHub

詳見 `docs/plan.md`(歷次決議)、`docs/progress.md`(進度狀態)。
