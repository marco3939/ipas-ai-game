請逐題確認題目品質:

### q_0001(Recall 公式 — errata)
```
題:下列何者為 Recall(召回率)的正確公式?
選項:
  TP/(TP+FP)        ← 與 Precision 混淆
  TP/(TP+FN)        ← ✅ 正解
  TP/(FN+TN)        ← 杜撰
  TP/(FP+FN)        ← 杜撰
解釋:Recall 衡量真為正中被找出比例,分母 = TP+FN
記憶口訣:Recall 看實際,Precision 看預測
```
✅ 結構完整、選項品質高、有勘誤標註

### q_0006(F1 計算 — 鐵律 #2 動態題)
```
4 個 case 隨機抽:case_a (P=0.8,R=0.6) 答 0.686
公式 F1 = 2PR/(P+R) = 2×0.8×0.6/1.4 = 0.686
變化型:每場遊戲可能換 case_b/c/d
```
✅ 鐵律 #2 完美實現

### q_0017(過擬合對策陷阱題)
```
題:不屬於降低模型複雜度的過擬合對策?
✅ 正解:擴增輸入特徵變數(反向操作)
陷阱:選項都是過擬合對策,只有一項是反向
```
✅ 高品質陷阱題,直接對應 114-2 Q26

### q_0040(VAE/GAN/Diffusion — 高優先預測)
